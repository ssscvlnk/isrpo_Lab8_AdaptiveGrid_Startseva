## Теория. Введение в Grid
**CSS Grid** — это мощная двумерная система компоновки, которая позволяет
создавать сложные сетки по строкам и столбцам.
Grid идеально подходит для:
- Создания полноценных макетов страниц
- Сложных раскладок с несколькими областями
- Галерей изображений
- Дашбордов и административных панелей
Основные концепции:
- **Grid-контейнер** — родительский элемент с display:grid
```
.grid-container {
    display: grid;
}
```
- **Grid-элементы** — дочерние элементы внутри контейнера
```
<div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
</div>
```
- **Grid-линии** — разделительные линии между ячейками
```
.grid-container {
    grid-template-columns: 200px 200px 200px;
}
```
- **Grid-ячейки** — пересечение строки и столбца, пример сетки 3х3:
```
.grid-container {
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: repeat(3, 1fr);
}
```
- **Grid-области** — прямоугольные области из нескольких ячеек
```
.grid-container {
    grid-template-areas:
        "header header"
        "sidebar main";
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
```

## Сравнение Flexbox и CSS Grid
### Общая идея
 | Критерий | Flexbox | CSS Grid |
|:-----|:-----|:-----|
| **Тип компоновки** | Одномерная  | Двумерная |
| **Управление** | Строка или столбец | Строки и столбцы |
| **Основное назначение** | Выравнивание элементов | Построение макетов |
| **Подходит для** | Компонентов | Страниц и сеток |

**Вывод**: Flexbox лучше подходит для **линейных структур**, Grid — для **табличных и сложных**
### Типичные сценарии использования
| Задача | Рекомендуется |
|:-----|:-----|
|**Навигационное меню**|Flexbox|
|**Карточки товаров**|Flexbox + Grid|
|**Центрирование элемента**|Flexbox|
|**Макет страницы**|Grid|
|**Галерея изображений**|Grid|
|**Дашборд**|Grid|
